Vim-Plug page:
https://github.com/junegunn/vim-plug
- install:
sh -c 'curl -fLo "${XDG_DATA_HOME:-$HOME/.local/share}"/nvim/site/autoload/plug.vim --create-dirs \
       https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim'
- note: remember open access to ~/.local/share/nvim and ~/.config/nvim

Install ripgrep to search text in files in telescope:
sudo apt-get install ripgrep

Install fd to search files in telescope:
sudo apt-get install fd-find

Lua languaje server:
Download the latest bin
https://github.com/sumneko/lua-language-server/releases
download it in: ~/.config/lsp/
create a folder named "lua-languaje-server" and decompress the file within

in bashrc or zshrc depend of the terminal engine, type:
export PATH="${HOME}/.config/lsp/lua-language-server/bin:${PATH}"

in nvim config type the setup in sumneko section from the server list:
https://github.com/neovim/nvim-lspconfig/blob/master/doc/server_configurations.md#sumneko_lua
